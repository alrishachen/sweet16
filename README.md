# Alrisha Sweet Sixteen

Standalone static copy of `https://alrishasweetsixteen.lovable.app/`.

## Run locally

From this folder:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deploy

This site is plain static HTML/CSS/JS, so you can host it on:

- GitHub Pages
- Netlify
- Cloudflare Pages
- Vercel

Any of those can point a custom domain at this folder without needing the Lovable subscription.
